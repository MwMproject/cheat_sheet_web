JavaScript

Syntaxe / Méthode Description / Exemple
Base du langage
letDéclare une variable à portée de bloc (modulable).
Ex :let age = 25;
constDéclare une constante (valeur non réassignable).
Ex : const PI = 3.1416;
varAncienne syntaxe, portée fonctionnelle (à éviter en ES6+).
typeofRetourne le type d’une variable.
Ex : typeof "texte" → "string"
console.log()Affiche une valeur dans la console
alert()Fenêtre d’alerte
prompt()Demande une saisie utilisateur
Types & opérateurs
+, -, *, /Opérateurs arithmétiques
===, !==Comparaison stricte / inégale
&&, ||, !Opérateurs logiques
Conditions & boucles
if / else if / elseCondition.
Ex : if (x > 10) {...}
switchTest de cas multiples
for / while / do...whileBoucles classiques
forEach()Itération sur un tableau
Fonctions
function nom() {...}Déclaration d’une fonction
const f = () => {...}Fonction fléchée (arrow function)
returnRetourne une valeur d’une fonction
Tableaux & objets
[]Tableau.
Ex : const fruits = ["pomme","banane"];
push() / pop()Ajoute ou retire un élément
map(), filter(), reduce()Manipulations avancées
{}Objet.
Ex : { nom: "Alice", age: 25 }
Object.keys()Retourne les clés d’un objet
DOM (Document Object Model)
document.querySelector()Sélectionne un élément
document.querySelectorAll()Sélectionne plusieurs éléments
element.textContentRécupère ou modifie le texte d’un élément
element.classList.add()Ajoute une classe CSS
addEventListener()Écoute un événement (click, input…)
Nouveautés ES2023+
Array.prototype.toSorted()Retourne un tableau trié sans modifier l’original
structuredClone()Copie profonde d’un objet
Promise.any()Renvoie la première promesse résolue
Web APIs utiles
fetch()Requête HTTP.
Ex : fetch("data.json").then(r => r.json())
localStorageStockage de données dans le navigateur
navigator.clipboard.writeText()Copie du texte dans le presse-papiers
IntersectionObserverDétection d’éléments visibles à l’écran
Asynchrone & temporisation
async function nom() {...}Déclare une fonction asynchrone.
awaitAttend la résolution d’une promesse.
Ex : await fetchData();
setTimeout()Exécute du code après un délai.
Ex : setTimeout(() => console.log("OK"), 1000);
setInterval()Répète du code à intervalle régulier.
PromiseObjet représentant une opération asynchrone.
Ex : new Promise(resolve => resolve("Terminé"));
.then() / .catch()Gestion des promesses réussies / échouées.
Modules & import/export
export function nom()Exporte une fonction depuis un fichier JS
import { nom } from "./module.js"Importe un module
